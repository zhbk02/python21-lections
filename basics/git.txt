git init  - создаем git репозитории из текущей папки 
git remote add название ssh- ссылка   - сохраняем ssh ссылку в какую  то переменную
git remote -v - посмотреть все сохраненные ssh ссылка
git add .  - просто пишем эту комманду каждый раз перед  commit
git commit -m 'message'  - создаем новую версию 
git push origin master  - отправляем в github
git pull origin master - стягиваем новые измеенения с github (если они есть в гихабе, но нет локально)
git clone ssh - ссылка  - копируем репозиторий с github (можно чужой )
git log - посмотреть все коммиты

git branch - показывает все ветки (* обозначена ветка на которой мы сейчас находимся)
git branch название  - создать новую ветку (просто создаем но не переходм)

git branch -D ветка  - удалить ветку

git checkout ветка  - перейти на ветку 
git checkout -b название  - созать ветки и перейти
